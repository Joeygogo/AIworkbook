# MyQuickQuiz

#### Video Demo:  [![Video Demo](https://img.youtube.com/vi/GuilWODFjAY/0.jpg)](https://youtu.be/GuilWODFjAY)

## Overview:
MyQuickQuiz is an innovative interactive learning platform designed to empower self-directed learners. By leveraging the power of AI-generated quizzes, this platform transforms your learning materials into engaging self-assessment tools. With a robust yet user-friendly interface, MyQuickQuiz streamlines file management and quiz generation, making it easier for users to track their progress and identify areas for improvement.

#### Description:
MyQuickQuiz is an interactive learning platform designed to enhance self-directed learning through AI-generated quizzes. Built using **JavaScript, SQLite3, Flask, Python, and Jinja**, this platform allows users to manage files, generate self-assessment quizzes, and review their learning progress efficiently. Whether you’re a student aiming to reinforce your understanding or a professional looking to stay sharp, MyQuickQuiz offers a flexible approach to self-assessment:

- **AI-Enhanced Learning:** By integrating ChatGPT’s capabilities, the platform dynamically generates a variety of quiz questions (multiple-choice, true/false, single-answer) tailored to your study materials. This intelligent question generation helps challenge your understanding and offers immediate feedback.
- **File Management:** The platform includes an intuitive dashboard where users can easily upload files and organize their study resources. File previews and an interactive workspace make navigating and selecting content simple and efficient.
- **Interactive Quiz Experience:** Upon selecting one or multiple files, users can generate quizzes with just a click. The system prevents immediate access to answers, ensuring you engage with the material first. After submitting your responses, you receive detailed explanations and correct answers to help solidify your learning.
- **User-Friendly Navigation:** The clean design incorporates a top navigation bar for quick access to key features (logo, logout, About), along with a left-side directory menu and a responsive right-side workspace.

## Features

### 1. Homepage & User Authentication
- A landing page with:
  - User **registration** and **login** functionality
  - An **About** section providing information about the platform
  - A **homepage link** for easy navigation
- Secure authentication system enabling users to register and log in safely.

### 2. User Dashboard
- Upon login, users are presented with a structured interface consisting of:
  - **Top Navigation Bar**: Displays product logo, logout option, and "About" button.
  - **Left-side Menu**: A directory structure where users can create folders and upload files.
  - **Right-side Workspace**: Provides file previews and interactive features.

### 3. File Management & Quiz Generation
- Users can **upload** and **organize files** in the left-side menu.
- Selected files can be **previewed** in the right-side workspace.
- A **self-assessment button** generates quiz questions based on selected files, using ChatGPT to create:
  - Multiple-choice questions
  - True/False questions
  - Single-answer questions

### 4. Quiz Interaction
- Users answer questions without immediate access to solutions.
- Upon **submission**, users receive the correct answers along with explanations.


## Technologies Used
- **Frontend**: JavaScript, HTML, CSS
- **Backend**: Flask (Python)
- **Database**: SQLite3
- **Templating Engine**: Jinja
- **AI Integration**: ChatGPT API

## Usage Tips

### 1. Organize Your Materials:
Create dedicated folders for different subjects or projects to keep your study resources well-organized. This makes it easier to generate targeted quizzes and monitor your progress.

### 2. Regular Self-Assessment:
Use the platform consistently to test your understanding of key topics. Reviewing the detailed feedback provided after each quiz can help you identify and address weak areas.

### 3. Leverage AI Capabilities:
Experiment with various file types and content formats to see how the AI generates different types of quiz questions. This flexibility can enhance your learning experience and keep the assessments challenging.
